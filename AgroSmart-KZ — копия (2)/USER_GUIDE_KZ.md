# AgroSmart KZ - Қолданушы нұсқаулығы

## Жылдам бастау

### 1. Орнату
1. `agrosmart-kz.zip` файлын жүктеп алыңыз
2. ZIP файлын ашыңыз
3. `index.html` файлын браузерде ашыңыз
4. Дайын! 🎉

### 2. Негізгі мүмкіндіктер

#### 📊 Жалпы шолу (Dashboard)
- **Не көресіз**: Негізгі метрикалар, графиктер, статистика
- **Қалай пайдалану**: 
  - Жыл мен облысты фильтрлеңіз
  - Графиктерге hover жасаңыз
  - AI ұсыныстарын оқыңыз

#### 🔍 Субсидия тексеру
- **Не істейді**: Өтінімді автоматты тексереді
- **Қалай пайдалану**:
  1. Облысты таңдаңыз
  2. Дақыл түрін таңдаңыз
  3. Ауданды енгізіңіз
  4. Субсидия сомасын енгізіңіз
  5. "Тексеруді бастау" батырмасын басыңыз
  6. Нәтижені қараңыз (Merit Score, тәуекел деңгейі)

#### 🌾 Жер талдауы
- **Не істейді**: Топырақ фотосын талдайды
- **Қалай пайдалану**:
  1. Топырақ фотосын жүктеңіз (2-5 сурет)
  2. Облысты таңдаңыз
  3. "Топырақ талдауын бастау" батырмасын басыңыз
  4. AI ұсыныстарын алыңыз

#### 📈 Болжамдау
- **Не істейді**: Келешек трендтерді болжайды
- **Қалай пайдалану**:
  1. Облыс пен бағытты таңдаңыз
  2. Болжам мерзімін (ай) орнатыңыз
  3. "Болжам жасау" батырмасын басыңыз
  4. Графикті қараңыз

#### 🗺️ Аймақтық карта
- **Не көресіз**: Облыстар бойынша деректер
- **Қалай пайдалану**:
  - Картадағы облыстарды қараңыз
  - Кестені скролл жасаңыз
  - Excel форматында жүктеп алыңыз

#### 🤖 AI Чат көмекші
- **Не істейді**: Сұрақтарға жауап береді
- **Қалай пайдалану**:
  1. Оң төменгі бұрыштағы чат белгішесін басыңыз
  2. Сұрағыңызды жазыңыз
  3. Enter батырмасын басыңыз
  4. Жауапты алыңыз

### 3. Жиі қойылатын сұрақтар

**С: Интернет қажет пе?**
Ж: Жоқ, барлығы локальды жұмыс істейді. Тек бастапқы жүктеу үшін қажет.

**С: Қандай браузерлерді қолдайды?**
Ж: Chrome, Firefox, Safari, Edge (соңғы нұсқалар).

**С: Деректерім қауіпсіз бе?**
Ж: Иә, барлық деректер сіздің компьютеріңізде сақталады.

**С: Мобильде жұмыс істей ме?**
Ж: Иә, толықтай адаптивті дизайн.

**С: AI шынымен жұмыс істей ме?**
Ж: Бұл демо нұсқа. Толық нұсқада нақты ML модельдері болады.

### 4. Қысқа пернелер

- `Esc` - Чатты жабу
- `Ctrl + K` - Іздеу
- `Tab` - Келесі өрісге өту

### 5. Техникалық қолдау

Мәселелер туындаса:
1. Браузерді жаңартыңыз
2. Кэшті тазалаңыз (Ctrl + Shift + Delete)
3. Басқа браузерді қолданып көріңіз
